## Project Overview
This project is a comprehensive Open Source Software (OSS) Audit focused on MySQL. It covers its history, licensing, ecosystem, Linux integration, and practical scripting implementations.

## Objectives
- Understand MySQL evolution
- Analyze GPL v2 licensing
- Explore LAMP stack role
- Practice Linux shell scripting
